## Js. tb en el escritorio :octocat:
==============

Introducción al desarrollo de aplicaciones javascript de escritorio, usando el framework Electron.

#### Ve los slides :speech_balloon:

  -`npm run start`

#### Ejemplos

  -hworld -> Hello world con Electron
  -hworld-nwjs -> Hello world con Nw.js
  -electroncomponents -> Menú contextual
  -videostreaming -> Uso de IPC para comunicar main y renderer
  -app-for-distribution -> Electron para distribución

#### Recursos :open_file_folder:

  - [Electron](http://electron.atom.io/)
  - [Awesome Electron](https://github.com/sindresorhus/awesome-electron)
  - [Electron-connect (live reload)](https://github.com/Quramy/electron-connect)
  - [Debug main process](https://github.com/atom/electron/blob/master/docs/tutorial/debugging-main-process.md)
